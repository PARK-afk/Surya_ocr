# OCR Processor

이미지에서 텍스트를 추출하는 OCR 처리 프로젝트

## 설치

1. 가상환경 생성 및 활성화
```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
# 또는
venv\Scripts\activate  # Windows
```

2. 패키지 설치
```bash
pip install -r requirements.txt
```

## 사용법

```bash
python main.py
```

## 기술 스택

- Python
- OpenCV
- Pillow
- Surya OCR
- PaddleOCR
